 TripPlanner
 Trip Planner - aplikacja do planowania wspólnych wyjazdów
 Borys Koza 95812 - Lider, backend / Szymon Chlastawa 95772 - frontend / Maks Kortus 96836 - tester, chmura
 Opis projektu
Trip Planner to aplikacja webowa umożliwiająca wspólne planowanie wyjazdów ze znajomymi. Użytkownicy mogą tworzyć wyjazdy, zapraszać innych uczestników, planować atrakcje oraz zarządzać wspólnym budżetem. Celem aplikacji jest ułatwienie organizacji podróży w grupie poprzez zebranie wszystkich potrzebnych informacji w jednym miejscu.

Dodatkowo aplikacja będzie zawierała prostego asystenta AI, który pomoże użytkownikom w planowaniu podróży. Asystent będzie analizował plan wyjazdu i podpowiadał czy o czymś nie zapomniano, np. o rezerwacji noclegu, transporcie lub ważnych elementach planu.

 Główne funkcjonalności
- tworzenie i zarządzanie wyjazdami  
- zapraszanie użytkowników do wspólnej podróży  
- planowanie atrakcji i planu dnia  
- lista miejsc do odwiedzenia  
- dzielenie kosztów między uczestników  
- wspólna lista rzeczy do spakowania  
- sugestie i przypomnienia generowane przez AI

 Architektura
Aplikacja będzie oparta na architekturze mikroserwisowej.

Planowane mikroserwisy:
- **User Service** - rejestracja i logowanie użytkowników  
- **Trip Service** - zarządzanie wyjazdami, planem podróży i kosztami  
- **AI Assistant Service** – sugestie i podpowiedzi dotyczące organizacji wyjazdu
